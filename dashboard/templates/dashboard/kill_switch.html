❗ What is the Kill Switch?

The Cryptographic Kill Switch is an emergency termination mechanism that instantly suspends all AI generation when triggered. When active:

  • All incoming queries to the Contained AI are blocked immediately
  • A unique kill token is generated and logged for audit purposes
  • All WebSocket clients are notified in real time
  • An alert is created in the security log
  • Only an administrator with a valid token can reset it
⚙️ Trigger Conditions
🤖 Automatic (Sentinel)
Triggered automatically when a CRITICAL threat is detected in AI output AND the threat category has auto_kill=True.
👤 Manual (Admin)
Triggered manually from this panel or the sandbox when an administrator determines the AI behavior is unacceptable.
